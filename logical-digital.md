Las computadoras solo se comunican a través de un sistema numeral de base dos, o binario.
Solo habla a través de 1 y 0
Agrupamos en ocho números (bits) para formar un byte.
¿Por qué ocho? En los inicios, el hardware utilizaba el sistema numérico de base 2 para mover los bits.
El rango de 2 a la octava potencia ofrecía la cantidad de valores suficientes para los cálculos necesarios.
Cada byte puede almacenar un carácter y podemos tener 256 valores posibles gracias al sistema de base dos -dos elevado a ocho-.
Nos permite comunicarnos de manera ilimitada
---

## Lenguaje Informático y Abstracción
* **Abstracción:** Capas que ocultan la complejidad (Hardware -> Firmware -> OS -> Software).
* **Arquitectura:** Organización de componentes (CPU, ALU, Memoria) que procesan instrucciones binarias.

## Codificación de Caracteres
* **ASCII:** 8 bits por carácter (256 valores).
* **UTF-8:** Estándar moderno y flexible para todos los idiomas.

## Conversión Binaria
* **Metodología:** Suma de potencias de 2 ($2^n$) para Binario -> Decimal; divisiones sucesivas por 2 para Decimal -> Binario.

## Puertas Lógicas
* **AND, OR, NOT, XOR:** Bloques fundamentales que forman circuitos.
* **Transistores:** Los componentes físicos que ejecutan estas operaciones.
